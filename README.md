## Getting Started

1. Install Hexo cli
```
$ npm install hexo-cli -g
```

2. Install hexo package
```
$ npm install
```

3. Generate static files
```
$ hexo generate
```

4. Start a local server
```
$ hexo server
```

5. Deploy to github.io
```
$ hexo deploy
```
